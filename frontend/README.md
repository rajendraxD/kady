# KADY Hiring Portal (basic)

A minimal React + Vite scaffold for the KADY Hiring Portal.

Quick start:

```bash
cd "kady-hiring-portal"
npm install
npm run dev
```

Open the site at the address Vite prints (usually http://localhost:5173).

Files created:
- package.json
- index.html
- src/main.jsx
- src/App.jsx
- src/index.css
- src/components/JobList.jsx
- src/components/JobForm.jsx
